# Sabores Bolivianísimos

## 🧠 Descripción
Aplicación móvil que permite descubrir y disfrutar la gastronomía boliviana. Incluye recetas auténticas, modernas y un buscador inteligente por ingredientes disponibles.

## 🎯 Objetivo
Conectar a las personas con la cultura culinaria boliviana a través de una app accesible, que aprovecha lo que ya tienen en casa para sugerir recetas tradicionales.

## ⚙️ Stack
> *⚠️ Supuesto: pendiente de definir. Probable stack mobile.*

| Tecnología | Uso |
|---|---|
| React Native / Flutter | App móvil |
| Node.js / Next.js | Backend API |
| PostgreSQL | Base de datos de recetas |
| AI/NLP | Búsqueda por ingredientes |

## 🌟 Features Principales
- Búsqueda de recetas por ingredientes disponibles
- Catálogo de recetas bolivianas auténticas
- Versiones modernas y adaptadas de platos tradicionales
- Filtros por región, tipo de plato, tiempo de preparación
