# LaMovidaBO

## 🧠 Descripción
Galería web interactiva de artistas urbanos bolivianos con estética visual basada en **pixel art**. Muestra artistas con animaciones hover dinámicas e integración con Spotify.

## 🎯 Objetivo
Crear un espacio digital único para la escena urbana boliviana, diferenciado por su estética pixel art y experiencia de usuario inmersiva.

## ⚙️ Stack

| Tecnología | Propósito |
|---|---|
| Next.js | Framework principal frontend |
| TypeScript | Lenguaje principal |
| CSS Modules + Tailwind | Estilos modulares |
| ShadCN | Componentes UI |
| Framer Motion | Animaciones fluidas |
| Aseprite | Creación de gráficos pixel art |
| Vercel | Hosting y CI/CD |
| Hostinger | Gestión de dominio |

## 🎨 Concepto Visual
- **Estética:** Pixel art retro con colores vibrantes
- **Animaciones:** Hover dinámicos por artista
- **Integración:** Spotify para contenido multimedia por artista

## 🎯 Público Objetivo
- 18-35 años
- Amantes de la música urbana boliviana
- Cultura digital y arte pixelado
