# 🌟 Concepto del Proyecto
- **Título**: *LaMovidaBO*
- **Objetivo**: Crear una **galería web interactiva** de artistas urbanos bolivianos con un estilo visual único basado en **pixel art**.
- **Diferencial**: 
  - Animaciones hover dinámicas.  
  - Integración con Spotify para mostrar contenido multimedia.

---

# 🎯 Público Objetivo
- **Edad**: 18-35 años.  
- **Intereses**:  
  - Música urbana.  
  - Cultura digital.  
  - Arte en estilo pixelado.

---

# 🛠️ Tecnologías Utilizadas
| Tecnología       | Propósito                                  |
|------------------|-------------------------------------------|
| **Next.js**      | Framework principal para el frontend.     |
| **TypeScript**   | Lenguaje de programación principal.       |
| **CSS Modules**  | Estilos modulares y aislados.             |
| **Tailwind CSS** | Utilidades de estilo modernas.            |
| **ShadCn**       | Componentes estilizados y dinámicos.      |
| **Framer Motion**| Animaciones fluidas y atractivas.         |
| **Aseprite**     | Creación de gráficos en pixel art.        |
| **GitHub**       | Repositorio del proyecto.                 |
| **Vercel**       | Hosting y despliegue continuo.            |
| **Hostinger**    | Gestión del dominio del sitio web.        |
