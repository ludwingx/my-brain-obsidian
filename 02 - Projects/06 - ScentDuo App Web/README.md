# ScentDuo App Web

## 🧠 Descripción
Sistema monolítico completo para e-commerce de perfumería. Combina una landing page pública (catálogo) con un sistema administrativo robusto (oculto) para gestión comercial e inventario.

## 🎯 Objetivo
Unificar la vitrina digital del negocio con sus operaciones de back-office, permitiendo ventas ágiles vía WhatsApp y control de stock centralizado para tienda premium de fragancias.

## ⚙️ Stack

| Tecnología | Propósito |
|---|---|
| Next.js 16 (App Router) | Framework frontend/backend híbrido |
| TypeScript | Lenguaje de tipado estricto |
| Prisma ORM | Acceso a dabatase |
| PostgreSQL / MySQL | Motor de base de datos |
| Auth.js (NextAuth) | Gestión de sesiones y seguridad |
| Tailwind CSS + Shadcn | UI moderna y minimalista |

## 🔑 Características Principales
- **Landing Pública:** Catálogo con filtros, descripción de notas olfativas, contacto por WhatsApp.
- **Panel Administrativo Oculto:** Ruta de acceso en URL específica, protegida por middleware.
- **Gestión:** Productos (categorías, tamaños, notas), Inventario (entradas/salidas), Estadísticas.
- **Diseño UI/UX:** Sobrio, elegante (negro, dorado, beige), mobile-first.
