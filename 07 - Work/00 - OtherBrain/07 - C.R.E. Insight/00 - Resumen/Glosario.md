# Glosario de Términos - C.R.E. Insights

## A

**API (Application Programming Interface)**  
Interfaz que permite la comunicación entre diferentes aplicaciones o sistemas, como n8n y Apify en este proyecto.

**Alertas**  
Notificaciones generadas por el sistema cuando se detectan publicaciones negativas o menciones críticas.

## B

**Backend**  
Parte del sistema que procesa datos, gestiona bases de datos y APIs. En este proyecto, n8n se usa como backend.

**Base de Datos (DB)**  
Sistema para almacenar información estructurada de posts, métricas y menciones externas. Se usa PostgreSQL junto con Prisma.

## D

**Dashboard**  
Panel visual donde se muestran métricas de engagement, publicaciones negativas y menciones externas.

## E

**Engagement**  
Indicador de interacción de los usuarios con publicaciones (likes, comentarios, compartidos).

**Etiquetado Manual**  
Proceso de clasificación de publicaciones de manera manual para entrenar el análisis de sentimiento.

## N

**N8N**  
Herramienta de automatización de workflows que permite integrar scraping, APIs y procesamiento de datos.

**Novatez**  
Parte del proyecto que indica dependencias o limitaciones iniciales, como el scraping manual.

## P

**PostMetrics**  
Métricas asociadas a cada publicación, como likes, comentarios, shares y sentimiento.

**Pipeline de Procesamiento**  
Secuencia de pasos para procesar datos obtenidos, incluyendo análisis de sentimiento y categorización.

**Prisma**  
ORM usado para manejar la base de datos PostgreSQL de forma estructurada.

## S

**Scraping**  
Proceso de extracción automática de datos de redes sociales (Facebook, Instagram, TikTok) usando Apify.

**Sentimiento (Análisis de Sentimiento / NLP)**  
Clasificación de publicaciones según su carga emocional (positiva, negativa, neutral).

## T

**TikTok**  
Red social donde se monitorean menciones de C.R.E. en perfiles y publicaciones.

**Tokens**  
Credenciales o claves usadas para autenticar y acceder a APIs de redes sociales.

## U

**Usuarios y Roles**  
Sistema de control de acceso que define qué funciones y datos puede ver cada usuario en el dashboard.
