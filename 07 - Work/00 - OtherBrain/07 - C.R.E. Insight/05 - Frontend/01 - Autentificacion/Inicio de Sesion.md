# Pagina de Inicio de Sesión

En esta pagina se visualizara un formulario de inicio de sesion, el usuario ingresara sus credenciales

**Credenciales:**
- Nombre de usuario
- Contraseña
Este formulario cuenta con 2 accesos de navegació:

- Iniciar sesión (Botón verde)
- "Regístrate" (Texto verde)
