## 🧠 Descripción
Conjunto de instrucciones y estructuración para el asistente (Agente IA) encargado de gestionar la arquitectura técnica y el desarrollo del ecosistema.

Mantener un flujo de trabajo continuo, modular y guiado por documentación como los Roadmap, System Flows y ADRs (Architecture Decision Records) dentro del repositorio del proyecto.
