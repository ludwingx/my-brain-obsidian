# Backlog, Roadmap e Ideas a Futuro

## 🏗️ Roadmap de Implementación

### Fase 1: Cimientos (Completado/En Progreso)
- [x] Estructura base Next.js 16 + Tailwind v4.
- [x] Modelado de base de datos en Prisma (Projects, Tickets, Users).
- [x] Sistema de autenticación base.

### Fase 2: Ejecución Core (Prioridad Actual)
- [ ] Implementación de Kanban por módulos.
- [ ] Motor de Time Tracking (Start/Stop).
- [ ] Dashboard de métricas para el CEO.

### Fase 3: Inteligencia Artificial y Colaboración
- [ ] Integración con OpenRouter para el "Ticket Architect".
- [ ] Sistema de notificaciones en tiempo real (Pusher o WebSockets).
- [ ] Portal para clientes externos.

## 💡 Ideas para el Futuro

- **Auto-Scheduler IA:** Que la IA no solo cree subtareas, sino que las asigne al desarrollador basándose en su carga de trabajo actual y especialidad detectada en tickets pasados.
- **Análisis de Rentabilidad AI:** Predecir si un proyecto se saldrá de presupuesto basándose en el ritmo de consumo de `realTime` frente al `estimatedTime`.
- **Visualizador de Esfuerzo:** Mapas de calor de actividad para identificar cuellos de botella en módulos específicos de código.
