# Vistas del Modulo Gestión de Pedidos

## Breadcrums: 
- se visualizan 2 breadcrums o navegadores entre paginas:
	- Panel de Control
	- Pedidos
## Cabecera: 
- Contiene un titulo principal y una descripcion de la pagina
- 2 botones:
	- Nuevo Pedido (Abré dialog para crear un nuevo pedido)
	- Historial de Pedido (Navegacion a page: Historial)
## Vista del sub-modulo Entregas para Hoy

En este modulo se pueden ver:
-Busquedas de pedidos
-Filtrar por estado de entrega
-Visualizar El numero de pedido, nombre del cliente, hora de la entrega, monto total, tipo de entrega, estado de la entrega y el el Boton de Ver Detalles

## Vista del sub-modulo Próximos a Entregar

En este submodulo se pueden ver: 
-Busquedas de pedidos.
-Filtrar por estado de entrega.
-Tabla de Proximas Entregas, que contiene columnas como:
- Numero de pedido
- Nombre del cliente
- Hora de Entrega
- Monto Total
- Tipo de Entrega
- Estado de Entrega
- Acciones (Contiene el botón de detalle)
