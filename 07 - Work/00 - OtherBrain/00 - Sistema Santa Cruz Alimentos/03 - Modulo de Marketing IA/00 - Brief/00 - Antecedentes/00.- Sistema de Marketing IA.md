# ⚙️ Workflow de n8n: Sistema de Generación de Contenidos con IA

> 🎯 **Objetivo**: Automatizar todo el ciclo de contenido —desde la planificación por IA, generación, revisión, hasta publicación— usando agentes encadenados en n8n.

---

## 🧩 Arquitectura general del sistema

- **Agente 1**: Estratega de contenidos (Planificación 📅)  
- **Agente 2**: Generador de contenido (Copys y guiones 📝)  
- **Agente 3**: Evaluador y revisor (Feedback y mejora 🔎)  
- **Agente 4**: Publicador automático (Redes, Email, Blog 📤)

---

## 🧠 Agente 1: Planificador de contenido

- **Rol**: Simula a un experto en contenido digital  
- **Tareas**:
  - Recibe pilares de contenido y objetivos de campaña
  - Genera un calendario de ideas con:
    - Fecha
    - Tema
    - Formato
    - CTA
    - Canal sugerido
- **Entradas necesarias**:
  - Lista de temas clave
  - Público objetivo
  - Frecuencia de publicación
- **Outputs**:
  - JSON o tabla tipo calendario
  - Guardado en Sheets / Airtable / Notion

---

## 🧠 Agente 2: Generador de contenido

- **Rol**: Redactor creativo con conocimiento técnico  
- **Tareas**:
  - Toma cada idea planificada del calendario
  - Genera texto para:
    - Post de Instagram
    - Newsletter
    - Blog
    - Scripts de video
  - Incluye hashtags y tono definido
- **Entradas necesarias**:
  - Idea planificada (tema, formato, objetivo)
  - Prompt de estilo y estructura
- **Outputs**:
  - Texto final formateado
  - Guardado en documento o API

---

## 🧠 Agente 3: Revisor y adaptador

- **Rol**: Crítico de contenido  
- **Tareas**:
  - Revisa la coherencia, ortografía, estructura
  - Ajusta tono y claridad
  - Toma feedback humano (opcional) o autoevalúa
- **Entradas necesarias**:
  - Texto generado
  - Criterios de evaluación (checklist o prompt)
- **Outputs**:
  - Versión final lista para publicar
  - Estado: aprobado/revisado

---

## 🧠 Agente 4: Publicador automático

- **Rol**: Community manager ejecutor  
- **Tareas**:
  - Publica automáticamente en:
    - Instagram (via API externa)
    - Email (SMTP o servicio)
    - Blog (API o CMS)
  - Registra logs y marca como publicado
- **Entradas necesarias**:
  - Contenido aprobado
  - Fecha programada
  - API Keys o tokens
- **Outputs**:
  - Confirmación de publicación
  - Log en Sheets / Notion

---

## 🧱 Recursos globales del sistema

- 🗂️ Base de datos de ideas planificadas  
- 🧠 Set de prompts para cada tipo de contenido  
- 📅 Scheduler (cron) para ejecutar cada agente en orden  
- 🔐 API Keys (OpenAI, SMTP, redes sociales)  
- 📊 Reportes y logs (opcional en Sheets o Notion)
