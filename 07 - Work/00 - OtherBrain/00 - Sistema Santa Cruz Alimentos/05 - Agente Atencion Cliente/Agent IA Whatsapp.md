##### **VPS**
Servidor Privado Virtual, es un servicio de alojamiento que utiliza la tecnología de virtualización para dividir un servidor físico en varios servidores virtuales más pequeños. Cada uno de estos servidores funciona de manera independiente, como si fuera un servidor físico por sí solo.
algunos vps's y sus planes:
- Google Cloud: 300$ free 90 dias
- Contabo: 4,76 $ mes
#### **EasyPanel**
Es un panel de control para servidores .
Se usa para **gestionar servicios, dominios, bases de datos y despliegues** de aplicaciones
##### **Evolution API**
Es una **API no oficial de WhatsApp** que permite enviar/recibir mensajes, gestionar chats y conectarse a WhatsApp Web sin necesidad de la API oficial de Meta
No requiere aprobación de Meta (a diferencia de WhatsApp Business API) y es más flexible para proyectos pequeños/medianos
