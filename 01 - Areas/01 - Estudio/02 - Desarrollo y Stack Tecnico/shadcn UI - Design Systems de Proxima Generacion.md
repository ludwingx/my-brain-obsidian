# 🎨 shadcn/ui: Estética Premium y Reutilización

Mi stack visual se basa en la flexibilidad. shadcn/ui no es una librería de componentes, es un **copy-paste sistemático** que me da control total.

## 🧠 Por qué lo uso en cada proyecto:
1.  **Radix UI Primitives:** La accesibilidad viene de caja. Modales, menús y desplegables funcionan perfectamente con teclado y lectores de pantalla.
2.  **Tailwind CSS (v4):** Estilos ultra rápidos sin salir del archivo. Uso de variables CSS para temas dinámicos (Dark Mode).
3.  **Framer Motion:** Integración de micro-animaciones para que la interfaz se sienta "viva".

## 🧩 Componentes Core que Domino:
- **DataTable:** El corazón de mis SaaS para mostrar proyectos y tickets.
- **Forms (React Hook Form + Zod):** Validación de extremo a extremo (Frontend + Server Actions).
- **Cards & Layouts:** Uso de `grids` complejos para dashboards.
