# ⚙️ Fundamentos de Desarrollo Backend

El Backend es el motor invisible que gestiona la lógica de negocio, la persistencia de datos y la seguridad.

## 🏗️ Pilares de un Backend Robusto

### 1. APIs y Protocolos
- **REST:** El estándar basado en recursos y verbos HTTP (GET, POST, PUT, DELETE).
- **GraphQL:** Para consultas flexibles y evitar over-fetching.
- **WebSockets:** Comunicación bidireccional en tiempo real (Chat, Subastas).

### 🔐 2. Seguridad y Autenticación
- **JWT (JSON Web Tokens):** Para sesiones stateless.
- **OAuth2 / OpenID Connect:** Integración con proveedores externos (Google, GitHub).
- **Hashing (BCrypt):** Protección de credenciales sensibles.

### 💾 3. Persistencia de Datos
Uso de ORMs (Prisma, Eloquent) vs Queries nativas. La elección depende de la complejidad y el rendimiento requerido.
