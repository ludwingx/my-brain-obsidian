# 🚀 Next.js: Maestría en Arquitectura App Router

El paso definitivo de mi aprendizaje: entender que la web debe ser rápida por defecto.

## 🧱 React Server Components (RSC)
Mi mayor aprendizaje fue entender que **React no es solo para el cliente**.
- **Server Components:** Lectura de base de datos directa sin sacrificar seguridad.
- **Client Components:** Solo para interactividad (botones, inputs, estados de formulario).
- **Directiva `'use client'`:** El marcador de frontera.

## 🔋 Server Actions & Mutaciones
Aprendí a ver el Backend como funciones asíncronas seguras, no solo como endpoints de API.
- Revalidación on-demand con `revalidatePath`.
- Optimistic Updates para que la UI se sienta instantánea.

## 💅 shadcn/ui: El Estándar de Componentes
No uso librerías pesadas. Uso la metodología de shadcn:
- **Radix UI:** Para la lógica de accesibilidad (modales, popovers).
- **Tailwind CSS:** Para el estilo atómico y flexible.
- **Copy & Paste:** Control total sobre el código del componente.
