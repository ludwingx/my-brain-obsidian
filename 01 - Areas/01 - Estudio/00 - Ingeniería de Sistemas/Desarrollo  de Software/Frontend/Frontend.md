# 🎨 Ingeniería Frontend Moderna

El Frontend es la intersección entre el código y la experiencia de usuario (UX).

## 🧩 Ecosistema de Desarrollo
1.  **Fundamentos Semánticos:** [[HTML]], [[JavaScript]], [[TypeScript]].
2.  **Estilo y Diseño:** CSS Moderno, Tailwind y animaciones.
3.  **Frameworks reactivos:** [[Frameworks/Frameworks|React, Next.js, Angular]].

## 🚀 Optimización y Rendimiento
- **Core Web Vitals:** LCP, FID y CLS. La velocidad es SEO.
- **Hydration:** El proceso de hacer interactivo el HTML estático.
- **Client-Side Rendering (CSR) vs Server-Side Rendering (SSR).**
