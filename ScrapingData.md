## 4️⃣ Publicación mixta (media combinada)

- Fotos + videos
- Slider dinámico

{  
  "type": "media_mixed",  
  "text": "...",  
  "media": [  
    { "type": "image", "url": "..." },  
    { "type": "video", "url": "..." }  
  ]  
}

👉 Este es el más complejo estructuralmente.

---

## 5️⃣ Publicación con enlace (Link Preview)

- Texto
    
- Preview enriquecido:
    
    - Imagen
    - Título
    - Descripción
    - Dominio

{  
  "type": "link_preview",  
  "text": "...",  
  "link": {  
    "url": "...",  
    "title": "...",  
    "description": "...",  
    "image": "..."  
  }  
}

⚠️ Importante porque el bloque HTML es diferente al de imagen normal.

---

## 6️⃣ Publicación compartida (Share)

Puede contener:

- Texto del usuario
- Post original embebido

{  
  "type": "share",  
  "text": "...",  
  "shared_post": {  
    "author": "...",  
    "text": "...",  
    "media": [...]  
  }  
}

👉 Aquí tienes post dentro de post.
